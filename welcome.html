Welcome to XatBrowser AI

Your AI-powered browser automation assistant

Getting Started

1. Configure AI Models

Create a config.json file in your C: drive with your AI model configurations:

{
  "azure-gpt4": {
    "service": "Azure",
    "modelName": "gpt-4",
    "maxTokens": 2000,
    "key": "your-azure-key",
    "temperature": 0.7,
    "endpoint": "your-azure-endpoint"
  },
  "claude-3": {
    "service": "Claude",
    "modelName": "claude-3",
    "maxTokens": 4000,
    "key": "your-claude-key",
    "temperature": 0.7
  }
}
2. Select a Model

Click the extension icon in your browser toolbar and select an AI model from the dropdown menu.

3. Start Automating

Type your request in the chat interface and let the AI assistant help you automate browser tasks.

Features

  • Natural language browser automation
  • Support for multiple AI models (Azure OpenAI and Claude)
  • Visual feedback for mouse movements and actions
  • Real-time validation of page elements
  • Interactive chat interface
Note: Make sure to keep your API keys secure and never share them. The extension reads the configuration file locally and does not transmit your keys to any external servers.

Example Commands

  • "Click the login button"
  • "Fill in the search box with 'Chrome extensions'"
  • "Navigate to google.com and search for 'AI automation'"
  • "Find and click all links containing 'download'"
